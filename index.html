Cloud Sync — GitHub
Connect to a private GitHub Gist for cross-device, cross-browser persistence. Your data syncs from any phone, tablet, or computer.
Create at github.com/settings/tokens → select only gist scope → Generate
If connecting a second device, paste the Gist ID shown after your first connect
How It Works
Cloud mode (GitHub)Data saved to a private Gist — accessible from any device/browser. Enter the same PAT + Gist ID on each device.
File mode (server.py)Data saved to trip_data.json locally. Run start.bat on your computer. Cross-browser on that machine.
Browser modeData in localStorage — per-browser only. Works offline, but doesn't sync across devices.
PriorityCloud → File → Browser. The console tries each in order and uses the first that works.
Setup for Mobile Access (One Time)
1. Host the HTMLPush Egypt_Umrah_Trip_Console.html to a GitHub repo → enable GitHub Pages → get your URL (username.github.io/repo)
2. Create a PATgithub.com/settings/tokens → New token (classic) → check only gist → Generate → copy it
3. ConnectOpen the console on any device → Sync tab → paste PAT → Connect. A Gist is created automatically.
4. Second deviceOpen the same URL → Sync tab → paste same PAT + the Gist ID from step 3 → Connect
One console — actions, card-spend targets, and trip expenses. Cloud sync via GitHub · Local via server.py · Browser fallback. Open the ⚙ Sync tab to configure.
