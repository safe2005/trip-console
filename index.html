Cloud Sync — GitHub
Connect to a private GitHub Gist for cross-device, cross-browser persistence. Your data syncs from any phone, tablet, or computer.
Create at github.com/settings/tokens → select only gist scope → Generate
If connecting a second device, paste the Gist ID shown after your first connect
Low Data Mode
Optimized for slow/cellular connections (e.g., 2G on iPhone). System fonts, throttled sync, effects off, lazy tab loading.
Low Data Mode

FontsSystem fonts instead of ~200 KB of web fonts (saves ~33s on 2G first load)

EffectsBackground texture, blur, and animations off
TabsLoaded only when opened (lazy render)
AutoOn Android, turns on automatically for 2G or Data Saver. iPhone has no connection API — use the manual toggle.
How It Works
Cloud mode (GitHub)Data saved to a private Gist — accessible from any device/browser, anywhere in the world. Enter the same PAT + Gist ID on each device.
Browser modeData in localStorage — per-browser offline fallback. Used only when GitHub is unreachable; syncs up to the cloud automatically once a connection is restored.
PriorityCloud → Browser. The console always prefers the cloud Gist and falls back to this browser only if GitHub can't be reached.
